## **Lumora – AI-Powered Women's Safety & Guardian Platform** 

## **The Idea** 

Personal safety applications often require users to manually trigger help during emergencies, which may not always be possible. Existing solutions often lack seamless communication between users, trusted guardians, and emergency services. 

**Lumora** is an AI-powered personal safety platform designed to provide proactive protection through Guardian Mode, live location sharing, SOS alerts, journey tracking, and intelligent emergency workflows. The platform enables trusted guardians to remotely monitor a user's safety while allowing users to quickly access help during emergencies.

## **Overview** 

Lumora is built using Flutter with Firebase powering authentication and cloud services. The application consists of multiple dashboards, including the Main Dashboard, Guardian Dashboard, Journey Tracking, and Safety Controls. 

Guardian Mode serves as the core safety feature. When enabled, trusted guardians can monitor the user's safety status and live location. Journey Tracking enables monitored trips with real-time location updates, while the SOS system remains accessible from every screen for immediate emergency assistance. 

The application follows a modern glassmorphism design language to deliver a clean, responsive, and intuitive user experience.
